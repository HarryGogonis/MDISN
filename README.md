# Requirements

* tweepy
* matplotlib
* pandas

# Usage

Please insert your keys into the files before continuing

* Collect streaming data
  `python stream.py > output.txt`
* Collect historical data
  `python collect.py Obama > obama.txt`
* Analyize data
  `python analyize.py output.txt`
